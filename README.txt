#Create short Url api

** Functionality: **

+ add Url to DB
+ create short path Url
+ visualization html page
+ redirect to Url by short link

** Tools: **
JDK 11, Spring , JPA/Hibernate, AngularJS, Lombok, JUnit 4,Maven 3, Git, H2 1.4, IntelliJIDEA.

** Notes: **
SQL init is located in the db/migration/V1__init.sql
